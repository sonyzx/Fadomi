# Fadomi
browser automation script for bruteforcing facebook .
## Gems needed 
```
gem install 'watir-webdriver'
gem install  'terminal-table'
gem install 'colorize'
```
## install chrome browser too
```
sudo gdebi google-chrome-stable_current_amd64.deb
```
